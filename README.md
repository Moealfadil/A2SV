# A2SV Coding Solutions 🚀

A collection of coding problem solutions from various competitive programming platforms, organized under the **A2SV** (Africa to Silicon Valley) organization.

## 📚 Platforms Covered

| Platform | Description |
|----------|-------------|
| [LeetCode](https://leetcode.com/) | Algorithm and data structure problems |
| [HackerRank](https://www.hackerrank.com/) | Programming challenges and tutorials |
| [Codeforces](https://codeforces.com/) | Competitive programming contests |
| [GeeksforGeeks](https://www.geeksforgeeks.org/) | DSA and interview preparation |

## 🛠️ Language

- **Python** 🐍

## 🌍 About A2SV

[A2SV](https://a2sv.org/) (Africa to Silicon Valley) is a social enterprise that enables high-potential university students to strengthen their problem-solving and coding skills.
